# Devcontainer Features

A collection of devcontainer features for cloud-native development tools.

## Features

| Feature | Description |
|---------|-------------|
| [gcloud-cli](src/gcloud-cli) | Google Cloud CLI with optional components |
| [aws-cli](src/aws-cli) | AWS CLI v2 with optional Session Manager and SAM CLI |
| [python-poetry-uv](src/python-poetry-uv) | Python with Poetry and uv package managers |
| [golang-lint](src/golang-lint) | Go with golangci-lint |
| [helm](src/helm) | Helm with optional plugins and Helmfile |
| [kubectl-kubectx-kubens](src/kubectl-kubectx-kubens) | kubectl, kubectx, kubens with optional Krew and K9s |
| [terraform](src/terraform) | Terraform with optional TFLint, Terragrunt, and tfsec |
| [argo-cli](src/argo-cli) | Argo CD, Workflows, and Rollouts CLI tools |
| [istio-cli](src/istio-cli) | Istio CLI (istioctl) |
| [cilium-cli](src/cilium-cli) | Cilium CLI with optional Hubble |

## Usage

Add features to your `devcontainer.json`:

```json
{
    "features": {
        "ghcr.io/Stardevs/devcontainer-features/gcloud-cli:1": {},
        "ghcr.io/Stardevs/devcontainer-features/kubectl-kubectx-kubens:1": {
            "installK9s": true
        },
        "ghcr.io/Stardevs/devcontainer-features/terraform:1": {
            "version": "1.9.0",
            "tflintVersion": "latest"
        }
    }
}
```

## Supported Platforms

- **Architectures:** amd64, arm64
- **Base Images:** Debian, Ubuntu

## Development

### Testing

```bash
# Install devcontainer CLI
npm install -g @devcontainers/cli

# Test a specific feature
devcontainer features test --features gcloud-cli

# Test with scenarios
devcontainer features test --features gcloud-cli --skip-autogenerated
```

### Publishing

Features are automatically published to GHCR when changes are pushed to the main branch.
